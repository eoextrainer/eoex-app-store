# DESIGN: Dunes Be One Basketball CMS Platform

## VISUAL DESIGN SYSTEM

### Color Palette

**Primary Colors:**
- Deep Orange (#FF6B35) - Main brand color, primary buttons, key highlights
- Black (#1A1A1A) - Headers, text, navigation background
- White (#FFFFFF) - Background, text contrast

**Secondary Colors:**
- Light Gray (#F5F5F5) - Secondary backgrounds, borders
- Dark Gray (#333333) - Secondary text
- Basketball Orange (#FFA500) - Accent, call-to-action elements

**Status Colors:**
- Success Green (#4CAF50) - Completed games, active status
- Warning Amber (#FFC107) - Pending, scheduled status
- Error Red (#F44336) - Cancelled, inactive status
- Info Blue (#2196F3) - Information, neutral status

### Typography

**Font Family:** Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

**Font Weights & Sizes:**
- Headers (H1): 32px, weight 700, line-height 1.3
- Headers (H2): 24px, weight 600, line-height 1.4
- Headers (H3): 18px, weight 600, line-height 1.4
- Body Text: 14px, weight 400, line-height 1.6
- Small Text: 12px, weight 400, line-height 1.5
- Button Text: 14px, weight 600, line-height 1.5

### Layout Principles

- **Tile-Based Design:** All content organized in 4-6 column responsive grid
- **Spacing:** 16px baseline grid (8px, 16px, 24px, 32px, 48px spacing)
- **Responsive Breakpoints:** Mobile (320px), Tablet (768px), Desktop (1024px), Large Desktop (1440px)
- **Alignment:** Left-aligned text, centered titles, right-aligned actions

## RESPONSIVE DESIGN APPROACH

### Breakpoints & Adaptation Strategy

**Mobile (320px - 767px):**
- Single column layout
- Stacked tiles
- Hamburger menu for navigation
- Larger touch targets (48px minimum)
- Simplified tables (horizontal scroll)

**Tablet (768px - 1023px):**
- Two-column layout
- Side navigation drawer
- Medium-sized tiles (2 per row)
- Optimized table views
- Touch and mouse input support

**Desktop (1024px+):**
- Multi-column grid layout (4-6 columns)
- Fixed sidebar navigation
- All features visible without scrolling (where possible)
- Mouse-optimized interactions
- Detailed views available

## INTERACTION PATTERNS

### Button States

**Primary Button (Orange #FF6B35):**
- Normal: Solid orange, white text
- Hover: Darker orange (#E55A2B), white text
- Active: Solid orange with shadow
- Disabled: Light gray (#CCCCCC), darker gray text

**Secondary Button (Black #1A1A1A):**
- Normal: Black, white text
- Hover: Dark gray (#333333), white text
- Active: Black with shadow
- Disabled: Light gray, darker gray text

### Form Elements

- Input fields: Light gray (#F5F5F5) background, 1px dark gray border
- Labels: Dark gray (#333333) text, 12px bold, above input
- Error states: Red (#F44336) border, red error message below
- Success states: Green (#4CAF50) checkmark, subtle green highlight

### Navigation

- Top navbar: Black (#1A1A1A) background, white text, fixed
- User menu: Dropdown from top-right, white background, black text
- Breadcrumb: Gray text (12px), forward slash dividers
- Active navigation: Orange (#FF6B35) underline or background

### Data Tables

- Header row: Light gray (#F5F5F5) background, bold text
- Rows: White background, 1px light gray bottom border
- Hover: Light gray (#F5F5F5) background on row hover
- Sorting: Orange arrow indicator (â†‘â†“)
- Pagination: Standard numbered pagination, orange active page
