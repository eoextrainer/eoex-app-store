# Dunes Be One - Basketball CMS Platform

## Project Overview

A comprehensive Content Management System (CMS) for managing basketball player training, games, statistics, and performance metrics. Built with Python (Flask), MySQL, HTML, CSS, and JavaScript SPA architecture.

**Organization:** Dunes Be One Basketball Organization

## Quick Start

```bash
# Clone the repository
git clone <repo_url>
cd Dunes

# Run with Docker
docker-compose up --build

# Access the application
http://localhost:5000
```

## Directory Structure

```
Dunes/
├── backend/              # Python Flask API
├── frontend/             # HTML, CSS, JavaScript SPA
├── database/             # MySQL schema and CSV data
├── docker/               # Docker configuration
├── data/                 # CSV data files
├── docs/                 # Documentation and diagrams
└── README.md
```

## Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (SPA)
- **Backend:** Python 3.9+, Flask
- **Database:** MySQL 8.0+
- **Containerization:** Docker & Docker Compose
- **Version Control:** Git with proper branching strategy

## Main Features

1. **Athlete Management** - Training, health, performance tracking
2. **Game Management** - Schedules, results, statistics
3. **Performance Analytics** - Player statistics and performance metrics
4. **Multi-role Access** - Athletes, Coaches, Clubs, Club Managers
5. **News Management** - Basketball news and updates
6. **Data Export** - CSV export for archiving

## User Roles

- **Athletes** - Track personal performance and training
- **Coaches** - Manage team rosters and player development
- **Clubs** - Manage teams and schedules
- **Club Managers** - Executive dashboard and analytics
