# EOEX Store Documentation

Welcome to the EOEX Store project!

## Overview
EOEX Store is a Netflix-style hybrid mobile and web app built with React Native and Expo. It features a cinematic UI, secure authentication, deep linking to YouTube, and is fully dockerized for easy deployment.

## Features
- Netflix-inspired dark UI and animations
- Cinematic splash and home screens
- Secure authentication and session management
- Deep linking and inline YouTube player
- Hybrid mobile/web support (Expo/React Native)
- Dockerized for easy deployment

## Getting Started
1. **Clone the repository:**
   ```bash
   git clone git@github.com:eoextrainer/eoex-app-store.git
   ```
2. **Open in GitHub Codespaces or locally:**
   - Codespaces: Click the green "Code" button on GitHub and select "Open with Codespaces".
   - Local: Open in VS Code and use the provided Dockerfile/devcontainer.
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the app:**
   ```bash
   npx expo start
   ```

## Project Structure
- `/src` - App source code
- `/docs` - Project documentation (this site)
- `/Dockerfile` - Docker configuration
- `/.devcontainer` - Codespace configuration
